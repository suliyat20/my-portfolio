# Portfolio Website Clone

This project recreates a personal portfolio website using HTML, CSS, and Javascript.
The website presents personal information, services, projects, and contact details.

## 📜 Features
- Navigation Menu
- Hero introduction section
- About section with skills
- Services section
- Portfolio work gallery
- Contact form
- Responsive layout for mobile devices
